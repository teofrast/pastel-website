# Сайт-визитка психолога-психотерапевта

Лендинг для частной практики **Анастасии Пайкиной** — психолога и психотерапевта, работающего онлайн.

Сайт построен как одностраничный лендинг (`index.html`) в тёплой кремово-бежевой пастельной гамме. Вёрстка самодостаточная, без сборочных шагов — открывается в браузере напрямую.

## Технологии

- **HTML5 / CSS3** — чистый статический лендинг, без JavaScript-фреймворков
- **Google Fonts** — Cormorant Garamond (заголовки) + Mulish (текст)
- **Node.js + [node-canvas](https://github.com/Automattic/node-canvas)** — генератор изображений-заглушек
- **WebP + PNG** — современные форматы картинок с фолбэком

## Разработка

### Требования

- **Node.js** ≥ 18 (только для генерации заглушек)
- Браузер (Chrome, Firefox, Safari, Edge)

### Установка и запуск

```bash
# 1. Установить зависимости (только для генератора заглушек)
npm install

# 2. Сгенерировать изображения-заглушки (если реальных фото ещё нет)
node generate-placeholders.js

# 3. Открыть сайт в браузере
#    Windows:  start index.html
#    macOS:    open index.html
#    Linux:    xdg-open index.html
```

Генератор создаст все необходимые изображения в папке `images/`, включая фавиконки и Open Graph превью.

### Просмотр палитр

```bash
start colors.html   # Windows
open colors.html    # macOS
```

## Процесс замены заглушек на реальные фото

1. Подготовить фотографии в нужных разрешениях (см. таблицу ниже)
2. Сохранить в `images/` с теми же именами (как PNG и WebP)
3. Удалить старые заглушки при необходимости

| Файл | Размер | Назначение |

| `hero.png` / `hero.webp` | 1600×878 | Портрет на первом экране |
| `hero-1200.png` / `hero-1200.webp` | 1200×658 | То же, 1200w |
| `hero-800.png` / `hero-800.webp` | 800×439 | То же, 800w |
| `about-me.png` / `about-me.webp` | 1023×1537 | Портрет в секции «Обо мне» |
| `quote-scene.png` / `quote-scene.webp` | 1600×600 | Фон секции с цитатой |
| `cta.png` / `cta.webp` | 1600×500 | Фон CTA-секции |
| `og-image.jpg` | 1200×630 | Превью для соцсетей |
